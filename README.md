# AutoImpressaoEtiquetas
## Introdução
A produção deste programa foi realizada com o intuito de automatizar e acelerar o processo de impressão de etiquetas, que quando realizado em larga escala torna-se um trabalho massante.
## Linguagens
Neste programa temos apenas a utilização de Python e algumas bibliotecas necessarias para o funcionamento do código.
Temos também a utilização do execel, possuindo uma planilha de produtos e outra apresentando a folha de impressão.
## Comentários 
O código possui comentários em algumas linhas explicando a função das própias. 
Na Linha (20) deveria ocorrer a impressão , mas como não possuo impressora, coloquei um 'print' simulando , mas o código parar impressão esta comentado.
